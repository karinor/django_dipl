## Install

#### Open console and clone repository to your local folder
```
$ git clone https://Dikyzz@bitbucket.org/Dikyzz/dpack.git
```

#### Change the directory
```
$ cd dpack
```

#### Install packages
```
$ npm install
```

#### Run gulp
```
$ gulp
```


## Usage

#### To run in development mode
```
$ gulp
```

#### To run in production mode(css & js minification)
```
$ gulp build
```
